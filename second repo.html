Today we learned how to connect vs code with github repiratory. To push codes to git hub, there are 3 things to do-

  1. git add .
  2. git commit -m"mention anything"
  3. git push

But to push codes to git hub at he first time, there are 6 things to do-

  1. echo "# my-2nd-repo" >> README.md
  2. git init
  3. git add .
  4. git commit -m "My Second Repo"
  5. git branch -M main
  6. git remote add origin https://github.com/Ruks02/my-2nd-repo.git
  7. git push -u origin main
